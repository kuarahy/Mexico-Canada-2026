## ✨ Features

| Feature | Description |
|---|---|
| 🗓️ **Full Schedule** | All 104 matches with official dates, kickoff times, and venues |
| 🕐 **Dual Time Zones** | Every match shows time in **GMT** and **CDT** |
| ⚽ **Live Score Entry** | Click any match to enter scores, track results, and set match status |
| 📊 **Auto Standings** | Group standings update instantly — Pts, W/D/L, GF, GD |
| 🏟️ **Knockout Bracket** | Full bracket from Round of 32 to the Final |
| 🏳️ **Flag Icons** | Every team displayed with their national flag |
| 🎨 **Flag Gradients** | Each group header uses that group's team flag colors |
| 💾 **Auto-Save** | Everything persists in your browser's `localStorage` — no server needed |

## 🚀 Getting Started

This is a **single-file, zero-dependency** dashboard. No installation. No build step.

```bash
# Clone the repo
git clone https://github.com/kuarahy/Mexico-Canada-2026.git
cd Mexico-Canada-2026

# Open in your browser
open worldcup2026.html        # macOS
start worldcup2026.html       # Windows
xdg-open worldcup2026.html    # Linux
```

> ⚠️ **Flags require internet access** — flag images are served from [flagcdn.com](https://flagcdn.com). Everything else works fully offline.

## 🗺️ How to Use

### Entering Results

1. **Click any match card** in the Group Stage, Knockout Bracket, or Schedule tabs
2. Enter the final score in the modal that appears
3. For knockout matches, penalty shootout fields appear automatically
4. Set the match **Status** (`Scheduled` / `Live` / `Completed`)
5. Click **Save** — standings and brackets update instantly

### Tabs

| Tab | What you'll find |
|---|---|
| ⚽ **Group Stage** | All 12 groups with match cards and live standings |
| 🏆 **Knockout** | The full bracket from Round of 32 through the Final |
| 🗓️ **Schedule** | Every match sorted by date with GMT/CDT times, filterable by stage |

### Schedule Filters

Use the filter buttons at the top of the Schedule tab to view:
- **All** matches
- **Group Stage** only
- **Knockout** only
- **Upcoming** (not yet played)
- **Completed** matches
